# RevitVersionChecker
A tool to check the version of Autodesk Revit

## Setup

1. Copy the `Tools` folder to the root of your C: drive:

2. Run `revitcheck.reg` to register the context menu.

> ⚠️ This file modifies the Windows Registry.  
> Please review its contents before running.

---

## Usage

After setup, you can launch the tool from the Windows context menu:

1. Right-click on a file
2. Click **RevitVersionChecker**
3. A dialog will display the current Revit version

4. Supported file types: `.rvt`, `.rfa`
